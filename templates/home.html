Python By Error

The Mission

Python by Error is a web textbook that takes an error-first approach to teaching Python. Python was my first programming language and when I started getting serious about it, errors freaked me out. The truth is, it took a long time before I realized how important they were to my self-actualization. What wisdom did I not gain! How much fun I did but didn't have?!?!? We're going to start off by dealing intimately with error messages so we can learn how to think in Python and become fearless in the process. Follow me down the stack trace!

A few features you should know about

The book contains exercises that can be done in the browser. The results for these are saved in your browser's local storage so you can save your progress over time. Additionally, for ease of use, use your keyboard's and keys to navigate between the sections.

Issues ?

If you find anything wrong with the book, errors, accessibility issues, etc., please open an issue on the Github issues page or send us a tweet at @PythonByError.
