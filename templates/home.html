      Python By Error

      Python by Error is a web textbook that takes an error-first approach to teaching python. Python was my first langauge, and when I started learning it, errors freaked me out. The truth is, it took a long time before I started reading them. How much time did I lose! How much fun did I not have?!?!? We're going to start off by dealing intimately with error messages so we can learn how to think in python and become fearless in the process. Follow me down the stack trace!
